Détails du projet

GREENVIEW

Cahier Des Charges

Langages : SQL, C#, Python

Durée : 80h

Conception d’un Modèle Conceptuel des Données (MCD) à partir d’un cahier des charges et réalisation d’un script de création et d’un script de remplissage de la base

Ecriture de fonctions python permettant d’afficher différents graphiques à partir de documents

Réalisation d’une interface graphique (IHM)

Création d’un script python faisant le lien entre l’interface et la base de données

Ecriture de différents rapports et réalisation d’un Gantt

Compétences Acquises

Travail en équipe

Gestion du temps

Meilleure compréhension des liens entre une base de données et un script python

Visualisation des données sur une carte et des graphiques

Découverte des besoins d’un client

Outils Utilisés

GoogleDocs : Pour rédiger les comptes rendus écrits

Win’Design : Pour la réalisation du MCD et du script de création de la base

SQL Server Management Studio : Pour la réalisation du script de remplissage de la base

Visual Studio Code : Pour la rédaction des fonctions pour les graphiques

Visual studio 2022 pour l’interface IHM
