LABYRINTHE

Cahier Des Charges

Langage : Java

Durée : 30h

Programmation d’un jeu de labyrinthe sur plusieurs étages avec un héros et des monstres

Création d'un diagramme UML final pour mieux visualiser les liens entre les classes

Rapport au format markdown qui explique le travail réalisé à chaque séance ainsi que les difficultés rencontrées et les solutions apportées

Compétences Acquises

Visualisation des actions du joueur

Meilleure compréhension des affichages graphiques

Compétences en java améliorées

Outils Utilisés

NetBeans : Pour le code du jeu et des tests unitaires

Gitlab : Pour enregistrer le travail

Visual Studio Code : Pour la rédaction du compte rendu au format markdown
