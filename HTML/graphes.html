Détails du projet

VISUALISATION DE GRAPHES

Cahier Des Charges

Langage : Java

Durée : 65h

Développement d'un logiciel pour visualiser et comprendre les données d'un graphe

Création d'un diagramme UML final pour mieux visualiser les liens entre les classes

Rapport qui rend compte des différents algorithmes implémentés et de leur complexité, et explique les méthodes utilisées

Compétences Acquises

Travail d'équipe

Visualisation de données sous formes de graphes

Meilleure compréhension des algorithmes de graphes

Compétences en java améliorées

Outils Utilisés

NetBeans : Pour le code du logiciel et des tests unitaires

Gitlab : Pour enregistrer le travail

GoogleDocs : Pour le compte rendu final
